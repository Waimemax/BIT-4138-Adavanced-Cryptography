# Secure Medical Equipment Staff Authentication System

## Project Overview

The Secure Medical Equipment Staff Authentication System is a web-based authentication platform designed to improve security and access control for personnel handling medical equipment. The system provides secure user registration, login authentication, role-based access control, and activity monitoring to ensure that only authorized staff can access sensitive medical equipment information and operations.

## Objectives

* Provide secure authentication for medical equipment staff.
* Prevent unauthorized access to sensitive resources.
* Implement role-based access control.
* Maintain audit logs of user activities.
* Enhance overall healthcare information security.

## Features

* User Registration
* Secure Login and Logout
* Password Hashing
* Role-Based Access Control
* Activity Logging
* User Management
* Secure Session Handling

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Waimemax/BIT-4138-Adavanced-Cryptography.git
```

2. Navigate to the project directory:

```bash
cd SecureMedicalAuthSystem
```

3. Create a virtual environment:

```bash
python3 -m venv venv
```

4. Activate the virtual environment:

```bash
source venv/bin/activate
```

5. Install dependencies:

```bash
pip install -r requirements.txt
```

6. Run the application:

```bash
python app.py
```
